# LIFs

**LIF** significa **Logical Interface**.

## Tipos

- **Node management**
- **Cluster management**
- **Data**
- **Intercluster**

## Ideia central

O IP é lógico. Ele não precisa ficar preso de forma rígida a uma interface física específica.

## Para memorizar

- node management: gerência do node;
- cluster management: gerência do cluster;
- data: acesso de clientes/hosts;
- intercluster: replicação e comunicação entre clusters.
