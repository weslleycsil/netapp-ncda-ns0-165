# IPspace, Broadcast Domain e Failover

## IPspace

Isolamento lógico de rede em nível IP. Uma boa forma de memorizar é pensar em algo parecido com VRF.

## Broadcast domain

Agrupamento de portas com características L2 compatíveis.

## Failover group

Conjunto de portas elegíveis para migração de uma LIF.

## Failover policies

- broadcast-domain-wide
- system-defined
- local-only
- sfo-partner-only
- disabled

## Pegadinha clássica

- broadcast domain não é a mesma coisa que IPspace;
- failover group não é a mesma coisa que failover policy.
