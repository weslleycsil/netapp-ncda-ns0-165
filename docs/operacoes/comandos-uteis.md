# Comandos úteis

## CLI geral

```bash
rows 0
set -d -c off
set -privilege admin
```

## Inspeção de cluster

```bash
cluster show
node show
storage failover show
```

## Rede

```bash
network interface show
network port show
network interface failover-groups show
```

## Storage

```bash
storage aggregate show
volume show
lun show
```
