# NCDA — NS0-165

Este material foi organizado como uma base de estudos para a certificação **NetApp Certified Data Administrator (NCDA)**.

A ideia aqui não é só listar tópicos, mas transformar o estudo em algo consultável e útil no dia a dia.

## Objetivos deste guia

- consolidar os conceitos centrais do ONTAP;
- organizar as anotações de aula/lab;
- servir como revisão rápida para a prova;
- virar uma base viva, que pode crescer com comandos, exemplos e cenários.

## Como usar

Sugestão de fluxo:

1. comece por **Fundamentos ONTAP**;
2. avance para **Storage** e **Networking**;
3. revise **Snapshots / SnapMirror**;
4. use **Perguntas rápidas** antes da prova.

## Prioridades de estudo

### Peso alto
- arquitetura ONTAP;
- aggregates, volumes e LUNs;
- snapshots;
- SnapMirror.

### Peso médio
- LIFs;
- IPspace / broadcast domain / failover;
- SVM;
- NAS vs SAN.

### Peso menor
- troubleshooting;
- hardware/boot;
- detalhes de linha de produto fora do ONTAP.
