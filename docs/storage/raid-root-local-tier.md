# RAID, root e local tier

## RAID

Os nomes mais comuns a revisar:

- **RAID-DP**: dupla paridade;
- **RAID-TEC**: tripla paridade.

## Root aggregate / vol0

- **aggr0**: aggregate de sistema;
- **vol0**: volume root do node.

## Local tier

Em versões mais novas, a NetApp usa com frequência o termo **local tier** como nomenclatura moderna para o conceito de aggregate.
