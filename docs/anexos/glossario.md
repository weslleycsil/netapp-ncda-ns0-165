# Glossário

- **Aggregate / Local tier**: pool físico de capacidade.
- **Volume / FlexVol**: container lógico.
- **LUN**: bloco apresentado ao host.
- **SVM**: entidade lógica de serviços de dados.
- **LIF**: interface lógica.
- **WAFL**: filesystem do ONTAP.
- **HA pair**: dupla de nodes com proteção mútua.
